This project is a simple browser-based music player that allows users to play, pause, and navigate through a playlist of songs. It provides a clean UI with album art, song details, and interactive controls, simulating a basic media player experience directly in the browser.

**⭐Project Link:-** https://dragorod.github.io/Music-Player/

**🛠️ Toolkits Used**
HTML
CSS
JavaScript

**⚙️ Development Process**
Built the layout for the player and playlist sections including buttons and song list.
Designed the interface using flexbox, custom variables, and responsive media queries.
Created a song dataset
Implemented play, pause, next, and previous controls
Managed state using a central object
Synced UI updates with audio playback
Enhancements
Highlighted the currently playing song
Added accessibility improvements 
Handled edge cases like reaching the end of the playlist

**📚 What I Learned**
Managing state without frameworks
Working with the Audio object for media playback
Structuring code into reusable and readable functions


**🚀 Features**
▶️ Play and resume songs
⏸ Pause and retain playback position
⏭ Next / ⏮ Previous track navigation
📃 Interactive playlist selection
🎯 Highlight currently playing song
📱 Responsive design for smaller screens
♿ Basic accessibility support (ARIA labels)
